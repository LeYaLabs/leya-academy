## 📖 Content

### 1. What is Hermes Agent

Hermes is an open-source framework that turns a terminal into a full AI assistant. It's not a chatbot — it's a **hub**. Think of it as an operating system for AI:

- Keeps memory between sessions
- Learns new skills without modifying source code
- Schedules automated jobs (cron)
- Connects to any AI model (DeepSeek today, local tomorrow)
- Runs 100% from the terminal — no browser, no GUI

You installed it on your machine. It's yours.

---

### 2. Installation

Dependencies are already resolved from L1.3 (Python 3.10+, Node.js 18+). You're ready.

**Install command:**

```bash
curl -fsSL https://install.hermes-agent.com | bash
```

> Check the official docs for the latest command: https://hermes-agent.nousresearch.com/docs

**Verify the installation:**

```bash
hermes --version
```

If you see a version number (e.g., `hermes 2.x.x`) → installation succeeded.

**Installation issues:**

| Issue | Solution |
|-------|----------|
| "command not found: hermes" | Restart the terminal or run `source ~/.bashrc` |
| "curl not found" | `sudo apt install curl -y` |
| Permission errors | Don't use `sudo` with the install command. Hermes installs as a regular user. |
| "unsupported node version" | Revisit Step 6 of L1.3 — upgrade Node.js to 20.x |

---

### 3. File structure

After installation, Hermes creates the `~/.hermes/` folder. Here's what's inside:

| Path | What it is | Importance |
|------|-----------|-----------|
| `~/.hermes/` | Main folder — everything lives here | — |
| `~/.hermes/.env` | API keys and sensitive settings | **Critical.** Never share it. Never put it on GitHub. |
| `~/.hermes/memories/MEMORY.md` | Assistant's memory — what it knows about the world | You edit it |
| `~/.hermes/memories/USER.md` | Your profile — who you are, what you prefer | You edit it |
| `~/.hermes/skills/` | Installed skills — extensions for Hermes | Auto-populated |
| `~/.hermes/plugins/` | Plugins — integrations with external platforms | Telegram, etc. |
| `~/.hermes/config.yaml` | Main configuration | You edit occasionally |

---

### 4. First configuration

```bash
hermes config init
```

Hermes will guide you interactively — answer the questions:

- **Assistant name:** pick what you want (e.g., "Nexus", "Jarvis", "Assistant")
- **Language:** `en` for English, `ro` for Romanian
- **Paths:** leave them as default (press Enter)

**Where providers go:** in `~/.hermes/.env`. You'll fill this in during L1.5 with your DeepSeek key.
