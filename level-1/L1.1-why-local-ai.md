## 📖 Content

### 1. Where is your AI right now?

In 2026, when you use ChatGPT, Claude, or Gemini, here's what happens:

- You type a question
- Your text travels over the internet to a server in Virginia, Frankfurt, or Singapore
- A massive AI model processes your question
- The answer comes back to you

Sounds simple. But:

- **You pay monthly.** $20/month for ChatGPT Plus. $240/year. Forever.
- **Your data sits on their servers.** Conversations, projects, ideas — all stored and analyzed.
- **If their server goes down, your AI disappears.** Nothing you can do. You wait.
- **You have no control.** Can't change how it thinks, what it remembers, how it responds.

The local model flips all of this.

---

### 2. Cloud vs Local — the real comparison

| | Cloud AI (ChatGPT, Claude) | Local AI (what you're building) |
|---|---|---|
| **Cost** | $20/month × 12 = $240/year | $0 — free software, only the API costs (~$0.50/million tokens) |
| **Your data** | On their servers | On your laptop |
| **Internet** | Required | Optional — Hermes works offline; only the API needs internet |
| **Control** | Zero | Total — you change anything |
| **Availability** | When their servers are up | 24/7, as long as your laptop is on |
| **Privacy** | Your conversations are training data | No one sees them, no one accesses them |

**Why does privacy matter?** Because anything you type into a cloud chatbot can be used to train the next version of the model. Your private conversation from yesterday = part of tomorrow's training data. On a local AI, your data never leaves your laptop.

### 4. 🧠 How It Really Works — LeYa Academy Architecture

> You might be wondering: "if everything is local, why do I need the DeepSeek API in Lesson 5?" Fair question. Let's clear it up.

**Hermes Agent is the hub, not the brain.**

Think of Hermes as an operating system for AI. It runs on your laptop — 100% local. But like any operating system, it can run applications that connect to the internet when needed.

| Component | Where it runs | What it does |
|-----------|--------------|--------------|
| **Hermes Agent** | 💻 Local (your laptop) | Orchestrates everything: memory, skills, conversations, cron jobs |
| **Your data** | 💻 Local (your SSD) | Conversations, preferences, memory — nothing leaves |
| **The AI model** | ☁️ API (DeepSeek) OR 💻 Local | Processes questions and generates answers |

**Why start with DeepSeek API (cloud) instead of a local model?**

1. **Simple.** No need to download a 2.4 GB model. No compiling llama.cpp. Just paste an API key.
2. **Fast.** Local models run slower on regular hardware. DeepSeek responds in 1-2 seconds.
3. **Capable.** DeepSeek V4 is one of the best models available. Small local models (2-3B parameters) are weaker.
4. **Cheap.** ~$0.50 per MILLION tokens — that's roughly 750,000 words processed. With $5 in starting credit, you'll have conversations for months.

**What's local and what's API? — Honest breakdown:**

| Is it local? | Component |
|-------------|-----------|
| ✅ Yes | Hermes Agent (the application) |
| ✅ Yes | Your memory (conversations, preferences) |
| ✅ Yes | Configuration files |
| ✅ Yes | Skills, cron jobs, watchdog |
| ☁️ API | DeepSeek (processes responses — Level 1 & 2) |
| ✅ Yes | Local model Gemma/Llama (processes responses — Level 3) |

**The long-term plan:**

```
Level 1: Hermes (local) + DeepSeek API (cloud) → fast, functional AI
Level 2: Hermes (local) + DeepSeek API → advanced customization
Level 3: Hermes (local) + LOCAL model → full autonomy, zero cloud
Level 4: ESP32, voice, multi-agent → complete 100% local ecosystem
```

**In short:** You start with the API to get going fast. Your data stays on your laptop — only the question text and the answer pass through DeepSeek (encrypted, via HTTPS). By Level 3, you can replace the API with a local model, and then EVERYTHING is on your machine. Zero data to the outside.

*If this sounds like an acceptable trade-off — keep going. If you want 100% local from day one — it's possible, but we recommend doing Levels 1-2 with the API and transitioning to local at Level 3.*
