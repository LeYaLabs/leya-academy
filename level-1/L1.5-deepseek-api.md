**Why DeepSeek V4 for this course?**

- **Unbeatable cost** — ~$0.50 per MILLION tokens (~750,000 words). Roughly 30x cheaper than OpenAI/Claude.

- **Stable API** — no sudden changes, clear documentation.
- **128K context** — you can have very long conversations without issues.
- **Globally available** — no geographic restrictions.

## 🔤 Glossary

| Term | What it means |
|------|---------------|
| **API key** | A long password that authorizes Hermes to use DeepSeek. Generated once. |
| **Token** | The unit of measurement for processed text. ~1 token = ~0.75 English words. |
| **Provider** | "Who processes it?" — DeepSeek in our case |
| **Model** | "Which version?" — `deepseek-chat` (V4) in our case |

## 📖 Content

### 1. Create an account and get an API key

1. Go to https://platform.deepseek.com
2. Click **Sign Up** — create an account with email or Google
3. After logging in, go to **API Keys** in the menu
4. Click **Create New Key** — give it a name (e.g., "Hermes")
5. **Copy the key immediately.** It's shown ONLY ONCE. Once you close the window, you can't see it again.

> Store the key somewhere safe. Don't post it anywhere. Don't send it via Discord/WhatsApp/email. Only one person should have access: you.

6. Add initial credit: **$5 is enough for months.** Go to Billing → Add Credit.

---

### 2. Configure in Hermes

**Add the API key:**

Edit the `.env` file:

```bash
nano ~/.hermes/.env
```

Add this line (replace `your-key-here` with what you copied):

```
DEEPSEEK_API_KEY=your-key-here
```

Save: `Ctrl+O`, Enter, `Ctrl+X`.

> **Alternative:** If you don't want to use nano, you can use `echo 'DEEPSEEK_API_KEY=*** >> ~/.hermes/.env`. But be careful — the key will appear in your terminal history.

**Activate the provider:**

```bash
hermes config set provider deepseek
```

**Set the model:**

```bash
hermes config set model deepseek-chat
```

> Why two commands? **Provider** = who processes it (DeepSeek). **Model** = which version (deepseek-chat = V4). You can change them independently anytime.

---

### 3. Your first conversation

```bash
hermes chat
```

You'll enter chat mode. Type:

```
Hello! Who are you and what can you do?
```

If the response is coherent and in the language you configured → **you did it.**

Exit chat: `Ctrl+C` or type `exit`.

**First conversation issues:**

| Issue | Solution |
|-------|----------|
| "Invalid API key" | The key is wrong or expired. Check platform.deepseek.com → API Keys. |
| "Provider not configured" | You didn't run `hermes config set provider deepseek`. Run it now. |
| "Connection timeout" | Check your internet. `ping platform.deepseek.com` — does it respond? |
| "Insufficient balance" | Add $5 credit at platform.deepseek.com → Billing |
| Response is gibberish/wrong language | Check your language setting: `hermes config get language`. Set to `en` or `ro`. |
